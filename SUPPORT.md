# Support

Need help with ufi?

- **Start with `ufi doctor --json`** — it checks host, API key, connectivity/TLS, and
  credential-file perms, and prints a fix for each failure.
- **Read the docs:** https://uficli.sh (or `ufi agent` for the embedded agent guide, and
  `ufi schema` for the machine-readable command/exit-code contract).
- **Questions / ideas:** open a [GitHub Discussion](https://github.com/rnwolfe/ufi/discussions).
- **Bugs:** open an [issue](https://github.com/rnwolfe/ufi/issues/new/choose) using the form
  (include `ufi version`, OS, the exact command, and `--json` output).
- **Security:** do **not** file a public issue — see [SECURITY.md](./SECURITY.md).

This is a community project maintained on a best-effort basis; there is no commercial SLA.
